Name of Quantlet: Heteroskedasticity_Tests_Corrections

Published in: Econometrics_R

Description: This R script illustrates a range of heteroskedasticity tests and correction methods in linear regression. Using the Engel food-expenditure data, it estimates an OLS regression of food expenditure on income, computes Breusch–Pagan and White LM tests via auxiliary regressions, and obtains White's heteroskedasticity-consistent (HC1) standard errors. It then demonstrates GLS with a known variance function and feasible GLS with an estimated variance function based on a log-variance regression. In a second example based on the CPS1985 wage data, the script implements a Goldfeld–Quandt-type test by comparing residual variances across gender groups and fits a groupwise FGLS model with gender-specific error variances.

Keywords: Econometrics, Heteroskedasticity, Breusch-Pagan test, White test, Robust Standard Errors, GLS, Feasible GLS, Goldfeld-Quandt test, Groupwise FGLS, R, lmtest, sandwich, quantreg, carData

Author: Jiajing Sun

Submitted: 22 November 2025
